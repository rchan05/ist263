Using Floats

This image is floated to the left, with text wrapped around it. Floating allows elements to be taken out of the normal document flow, so they appear to the left or right of surrounding content. Here, additional text flows alongside the image. We use a container with `overflow: auto;` to clear the float and prevent layout issues.

Adding a margin to the floated image keeps the text from sticking too closely to it, making for a more visually appealing layout. Floats are commonly used for image alignment, but Flexbox provides more control for modern layouts.
